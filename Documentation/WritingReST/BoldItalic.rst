=================
Bold, italic etc.
=================


bold
====

::

   This is normal text. **This is bold text**

How it looks:

This is normal text. **This is bold text**




italic
======

::

   This is normal text. *This is italic text.*

How it looks:

This is normal text. *This is italic text.*


Possible errors
===============

.. important::

   This does not work: Note the extra space between `**` and the beginning of the text.


::

   This is normal text. ** This is bold text**

How it looks:

This is normal text. ** This is bold text**
