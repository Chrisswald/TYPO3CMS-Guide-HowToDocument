==========================
Problems with inline style
==========================


If you use **bold** or *italic*, make sure there is no space between the markup
and the text.


Correct syntax
--------------

::

   This is normal text. **This is bold text.**

How it looks:


   This is normal text. **This is bold text.**

Wrong syntax
------------

::

   This is normal text. ** This is bold text.**

How it looks:


   This is normal text. ** This is bold text.**
